# POLY SHIP - DSA Assignment

## Giới thiệu

Project mô phỏng hệ thống logistics POLY SHIP.

Thuật toán sử dụng:

### Graph

- Dijkstra Shortest Path
- Kruskal MST
- DSU (Disjoint Set Union)

### Hashing

- Hash Table
- Group Anagrams
- Longest Consecutive Sequence
- Subarray Sum = K
- Rabin-Karp Rolling Hash

### Dynamic Programming

- Fibonacci Tabulation
- Climbing Stairs
- Knapsack 0/1 (2D)
- Knapsack 0/1 (1D)

## Độ phức tạp

### Dijkstra

O((V + E) log V)

### Kruskal

O(E log E)

### Hash Table

O(1) trung bình

### Longest Consecutive

O(n)

### Group Anagrams

O(n × k log k)

### Subarray Sum = K

O(n)

### Rabin-Karp

O(n + m)

### Fibonacci

O(n)

### Climbing Stairs

O(n)

### Knapsack

O(n × B)
